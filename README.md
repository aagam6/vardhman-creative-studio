# Vardhman Creative Studio

Premium React landing page for the official event website:

**परमवीर चक्र - शौर्यगाथा**  
Route: `/param-vir-chakra`

## Tech Stack

- React
- Vite
- Tailwind CSS
- Vanilla JavaScript behavior
- React Helmet for SEO metadata
- Lucide React icons

## Local Development

```bash
npm install
npm run dev -- --hostname 127.0.0.1 --port 3000
```

Open:

```text
http://127.0.0.1:3000/param-vir-chakra
```

## Build

```bash
npm run build --prefix apps/web
```

## Lint

```bash
npm run lint --prefix apps/web
```

## Page Highlights

- Premium patriotic design inspired by Indian flag colors, Amar Jawan Jyoti, Ashoka Chakra and National War Memorial ambience.
- SEO-first architecture with canonical URL, meta tags, Open Graph, Twitter cards and structured JSON-LD schemas.
- Mobile-first responsive layout.
- Accessible semantic sections, keyboard-friendly FAQ accordion and ARIA labels.
- Google Form based registration CTA.
- Event details, sacred inspiration, featured speakers, Param Vir Chakra heroes, why attend section, timeline, FAQ, organiser and contact sections.

## Deployment Notes

For Laravel/PHP deployment, build the React app and serve the generated static assets from the production public directory or integrate the built assets with the existing frontend pipeline.

```bash
npm run build --prefix apps/web
```

The production output is generated by Vite in the web app build directory.
